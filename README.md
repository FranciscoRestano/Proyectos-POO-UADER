# Proyectos-POO-UADER
Ejercicios de la carrera. Examenes, ejercicios, etc.
## Mini-Proyectos

Este repositorio demuestra la aplicación de Programación Orientada a Objetos (POO) y manejo de colecciones y archivos, conocimientos adquiridos en las materias de Algoritmos y POO.

### Proyectos Incluidos:

Gestor de Cursos: Sistema para calcular horas de cursos simples, correlativos y compuestos. Incluye funciones de Top 5 y persistencia de datos en `.txt` y binario.

Gestor de Telecomunicaciones: Sistema de entrada/salida para leer y organizar notificaciones (Mails y Textos) desde archivos binarios. Demuestra manejo de `structs` y clasificación de datos.

Gestor de Series y Películas: Sistema de valoración de videos. Incluye cálculo de valoración promedio y búsqueda del contenido más votado.
